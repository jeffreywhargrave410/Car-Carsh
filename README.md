### How to Use

1. Open `index.html` in your web browser
2. Click the "Start Crash Sequence" button to watch the trucks crash
3. Watch as the vehicles collide with sound captions like "CRASH!", "BOOM!", "BANG!", etc.
4. Click "Reset" to return to the initial state and watch again

### Running Locally

Simply open the `index.html` file in any modern web browser, or serve it with a local web server:

```bash
python3 -m http.server 8000
```

Then navigate to `http://localhost:8000` in your browser.
